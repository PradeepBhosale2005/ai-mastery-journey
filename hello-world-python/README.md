# Hello World Python Assignment

## Assignment

Set up a development environment, create a Git repository, write a simple "Hello World" program in Python, and add a basic README about the learning goals.

## Learning Goals

- Set up a basic Python development environment.
- Create and manage a Git repository.
- Write and run a simple Python program.
- Understand how to document a project using a README file.
- Prepare the assignment folder for LMS submission as a ZIP file.

## Requirements

- Python installed on the system
- Visual Studio Code or any code editor
- Git installed on the system

## Project Structure

```text
hello-world-python/
├── main.py
└── README.md
```

## How to Run This Project

### Step 1: Extract the ZIP File

If this project is submitted as a compressed ZIP file, first extract the ZIP file.

After extracting, open the folder:

```text
hello-world-python
```

### Step 2: Open Terminal

Open PowerShell, Command Prompt, or the terminal inside Visual Studio Code.

Go to the project folder:

```bash
cd hello-world-python
```

### Step 3: Run the Python Program

Run this command:

```bash
python main.py
```

If `python` does not work, try:

```bash
py main.py
```

```text
Hello, World!
```
