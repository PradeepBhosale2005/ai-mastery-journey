# Object-Oriented Programming Assignment

## Assignment Summary

This assignment contains Python solutions for Object-Oriented Programming concepts.

The exercises include:

1. Class Creation
2. Inheritance
3. Encapsulation
4. Polymorphism
5. Abstraction

## Learning Goals

- Create Python classes and objects.
- Use constructors to initialize object attributes.
- Apply inheritance by creating a subclass.
- Override methods in a subclass.
- Use encapsulation with private attributes and getter/setter methods.
- Demonstrate polymorphism by calling the same method on different object types.
- Use abstraction with an abstract base class and abstract methods.

## Requirements

- Python installed on the system

- Git installed on the system
