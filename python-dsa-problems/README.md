# Python DSA Problems Assignment

## Assignment Summary

This assignment contains Python solutions for 14 common coding problems from LeetCode and GeeksforGeeks. The problems cover arrays, strings, hashing, stacks, two pointers, number theory, and basic algorithmic thinking.

## Problem List

1. Contains Duplicate
2. Unique Prime Factors in Increasing Order
3. Fizz Buzz
4. Container With Most Water
5. Valid Palindrome
6. Longest Common Prefix
7. Convert Sentence to CamelCase
8. Longest Palindromic Substring
9. Group Anagrams
10. Two Sum
11. Top K Frequent Elements
12. Min Stack
13. Intersection of Two Arrays
14. Roman to Integer

## Learning Goals

- Practice solving array and string problems in Python.
- Use sets, dictionaries, counters, and lists effectively.
- Understand two-pointer techniques.
- Implement stack operations with minimum tracking.
- Write clean reusable functions.
- Add simple automated tests using Python unittest.

## Requirements

- Python installed on the system
- Visual Studio Code or any code editor
- Git installed on the system

## How to Run After Extracting ZIP File

### Step 1: Extract the ZIP File

Extract the submitted ZIP file on your system.

### Step 2: Open the Folder

Open the extracted folder in Visual Studio Code or any code editor.

### Step 3: Open Terminal

Open PowerShell, Command Prompt, or the integrated terminal in Visual Studio Code.

### Step 4: Go to the Assignment Folder

```bash
cd python-dsa-problems
```

### Step 5: Run All Solutions Demo

```bash
python solutions.py
```

If `python` does not work, try:

```bash
py solutions.py
```

### Step 6: Run Test Cases

```bash
python test_solutions.py
```

If `python` does not work, try:

```bash
py test_solutions.py
```

Expected result:

```text
OK
```
