## How to Run

Open terminal inside this folder and run any problem file separately.

Example:

```bash
python 01_contains_duplicate.py
```

Another example:

```bash
python 10_two_sum.py
```

If `python` does not work, try `py`:

```bash
py 01_contains_duplicate.py
```

## Notes

- Each file contains the problem statement summary.
- Each file contains one function or class solution.
- Each file has a small example under `if __name__ == "__main__"` so it can be run directly.
