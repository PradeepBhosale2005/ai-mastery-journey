# Pandas Assignment 05

## Module

Pandas Assignments

## Assignment Summary

This folder contains Python Pandas solutions for 10 assignment sections. Each assignment section is saved in a separate Python file.

It also includes an Iris dataset analysis notebook that demonstrates:

- Dataset cleaning
- Summary statistics
- 4 plots using Matplotlib

## Topics Covered

1. DataFrame Creation and Indexing
2. DataFrame Operations
3. Data Cleaning
4. Data Aggregation
5. Merging DataFrames
6. Time Series Analysis
7. MultiIndex DataFrame
8. Pivot Tables
9. Applying Functions
10. Working with Text Data
11. Iris dataset cleaning, summary statistics, and visualizations in Jupyter Notebook

## Requirements

- Python installed on the system
- Visual Studio Code or any code editor
- Pandas installed
- NumPy installed
- Matplotlib installed
- Jupyter installed for notebook execution

Install requirements using:

```bash
pip install -r requirements.txt
```

If `pip` does not work, try:

```bash
python -m pip install -r requirements.txt
```

## How to Run After Extracting ZIP File

### Step 1: Extract the ZIP File

Extract the submitted ZIP file on your system.

### Step 2: Open the Folder

Open the extracted folder in Visual Studio Code or any code editor.

### Step 3: Open Terminal

Open PowerShell, Command Prompt, or the integrated terminal in Visual Studio Code.

### Step 4: Go to the Assignment Folder

```bash
cd python-pandas-assignment-05
```

### Step 5: Install Requirements

```bash
pip install -r requirements.txt
```

If `pip` does not work, try:

```bash
python -m pip install -r requirements.txt
```

### Step 6: Run All Pandas Assignment Sections

```bash
python run_all.py
```

If `python` does not work, try:

```bash
py run_all.py
```

### Step 7: Run One Assignment File

Example:

```bash
python assignment_01_dataframe_creation_indexing.py
```

Another example:

```bash
python assignment_10_working_with_text_data.py
```

### Step 8: Run Test Cases

```bash
python test_pandas_assignment.py
```

If `python` does not work, try:

```bash
py test_pandas_assignment.py
```

Expected test result:

```text
OK
```
